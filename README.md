# Catatan Belajar Linux

Repositori ini berisi catatan pribadi selama mempelajari dasar-dasar sistem operasi Linux.

## Perintah Dasar Navigasi

- 'pwd' - menampilkan direktori aktif saat ini
- 'ls -la' - menampilkan semua file termasuk yang tersembunyi
- 'cd /path/to/dir' - berpindah direktori
- 'mkdir nama_folder' - membuat folder baru

## Manajemen File

- 'cp file1 file2' - menyalin file
- 'mv file1 /path/' - memindahkan file
- 'rm file1' - menghapus file
- 'cat file1' - menampilkan isi file

## Manajemen Proses

- 'ps aux' - melihat semua proses yang berjalan
- 'top' - monitoring proses secara real-time
- 'kill PID' - menghentikan proses berdasarkan PID

## Permission

- 'chmod 755 file' - mengubah permission file
- 'chown user:group file' - mengubah kepemilikan file

## Networking Dasar

- 'ip addr' - melihat IP address
- 'ping google.com' - mengecek konektivitas
- 'curl http://example.com' - mengambil konten dari URL
- 'ss -tuln' - melihat port yang terbuka
